Operator ProtobufBuild

Toolkits > com.ibm.streamsx.protobuf 1.0.0.2 > com.ibm.streamsx.protobuf > ProtobufBuild

The ProtobufBuild operator takes in a tuple of the type generated by spl-schema-from-protobuf script and converts it into a serialized Protobuf record. The output is a blob data type. The operator will find the first blob data type field in the output port schema and use it as the field to contain the serialized record.

The protoMessage parameter takes in the fully-qualified message name in the Protobuf schema (my.package.MyMessage). The protoDirectory parameter takes in the path (relative to application directory or absolute) to the directory containing the Protobuf files. The protoRootFile parameter takes in the path (relative to the protoDirectory) for the root Protobuf file to be processed. The operator will also include any files imported from this root file.
